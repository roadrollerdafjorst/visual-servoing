Install packages:
    pip install -r packages.txt

    
To run IBVS:
1.  Navigate to IBVS directory in terminal:
        in the terminal type
         cd IBVS        (if in src directory)
         cd ../IBVS     (if in PBVS directory)
2.  Run the code
        python3 IBVS.py


To run PBVS:
1.  Navigate to IBVS directory in terminal:
        in the terminal type
         cd PBVS        (if in src directory)
         cd ../PBVS     (if in IBVS directory)
2.  Run the code
        python3 pbvs.py
